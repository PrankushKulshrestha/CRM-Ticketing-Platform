# CRM Ticketing Platform

Full-stack CRM Helpdesk platform.

- `/BE` — Node.js / Express / MongoDB (TypeScript) backend
- `/FE` — React / TypeScript (Vite) frontend

## Setup

### Backend
```
cd BE
npm install
cp .env.example .env   # fill in your own values
npm run dev
```

### Frontend
```
cd FE
npm install
cp .env.example .env   # fill in your own values
npm run dev
```
